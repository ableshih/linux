粗體和斜體
1.8.1 說明
使用**或者__表示粗體。
使用*或者_表示斜體。

1.8.2 示例
 **粗体1**    __粗体2__
 *斜体1*    _斜体2_
粗體1 粗體2
斜體1 斜體2

1.8.3 注意
前後的*或_與要加粗或傾斜的字體之間不能有空格。
我通常在強調時使用加粗標記，在和一行中的加粗區分且也表示強調時使用傾斜標記，這裡的傾斜標記的使用場景不明確。如果你知道：請務必告訴我。










列表

# 標題
## 標題
### 標題
#### 標題
##### 標題
###### 標題
```
說明
使用 # 表示標題，
一級標題使用一個 #，
二級標題使用兩個 ##，
以此類推，共有 六級 標題。

範例
# 標題
## 標題
### 標題
#### 標題
##### 標題
###### 標題
```

这是高阶标题（效果和一级标题一样 ）
========
这是次阶标题（效果和二级标题一样）
--------------
