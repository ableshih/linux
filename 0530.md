## sudoer 不重開機
```
建一個 user

more /etc/passwd
可找到建出來的 user

sudo passwd user
設新user密碼

sudo su - user
切到新user 

id
看一下確認

sudo ls /etc/hosts
這不能使用sudo 指令

exit
切回有 root 權限使用者

id

查看sudoer權限列表
sudo more /etc/sudoers

把要有 sudo 的 user 加到列表中，當要下 sudo 系統會先查表

sudo su - user
切到新user 

id
看一下確認

sudo ls /etc/hosts
就可以使用sudo 指令
```
